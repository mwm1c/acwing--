# 766. 去掉多余的空格

输入一个字符串，字符串中可能包含多个连续的空格，请将多余的空格去掉，只留下一个空格。

输入格式
共一行，包含一个字符串。

输出格式
输出去掉多余空格后的字符串，占一行。

数据范围
输入字符串的长度不超过 200。
保证输入字符串的开头和结尾没有空格。

输入样例：
```
Hello      world.This is    c language.
```
输出样例：
```
Hello world.This is c language.
```


```c++
#include <iostream>

using namespace std;

int main()
{
    string s1, s2;
    getline(cin, s1);
    int flag = 0;
    
    for(int i = 0; i < s1.size(); i ++)
    {
        if(s1[i] != ' ')
        {
            if(flag == 1)
                flag = 0;
            s2 += s1[i];
        }
        else if(s1[i] == ' ' && flag == 0)
        {
            flag = 1;
            s2 += s1[i];
        }
        // else(s1[i] == ' ' && flag == 1)
        //     do nothing...
    }
    
    cout << s2 << endl;
    
    return 0;
}
```
